# AWSAndOAuth2
AWS를 이용해 배포 및 DB 연동, OAuth2 공부

## 목적
1. IntelliJ와 Spring Boot를 사용하여 IntelliJ 익숙해지기
2. 로컬 DB가 아닌 AWS RDS를 사용해 클라우드 DB 사용
3. OAuth2를 이용하여 소셜 로그인 기능 구현하여 OAuth2 개념 및 사용법 공부
4. AWS로 실제 배포해보기
